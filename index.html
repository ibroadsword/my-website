About

Hello, my name is Matthew Kirtley. I'm a self-taught web developer and currently am trying to learn the MEAN stack. This is a website to showcase any projects I've done, either for a course I'm taking, or something I've decided to make on my own. The CSS library utilized is w3.css as an alternative to Bootstrap.
